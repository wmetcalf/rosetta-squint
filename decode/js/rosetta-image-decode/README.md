# rosetta-image-decode (JS/TS)

Byte-exact PIL-compatible image decoder library, JS/TS port.

## Build + test

```
npm install
npm test
```

Tests resolve fixtures and goldens from `../../spec/` (relative to package root).

## v1 Formats

- BMP (Tier 1+2+3 minus BI_JPEG/BI_PNG)
